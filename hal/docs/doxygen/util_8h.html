unsigned char UTIL_DbguGetHexa32 ( unsigned int *  pValue)

Reads a hexadecimal number from the debug UART.

Warning
This is a blocking function which will cause the watchdog to reset the iOBC when the debug connector is not connected and will prevent the RTOS scheduler and other tasks from running
Parameters
[out]pValuePointer to storage location for input value
Returns
1 if successful, 0 if input is not a number or not a hexadecimal number
