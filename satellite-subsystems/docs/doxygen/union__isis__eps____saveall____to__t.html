Field Documentation

uint16_t _isis_eps__saveall__to_t::checksum

To force save this value can be set to 0. The save will then proceed without performing CRC verification.

uint8_t _isis_eps__saveall__to_t::conf_key

Configuration key: 0xA7. Any other value causes the save command to be rejected with a parameter error. The save will not be performed in that case.
