Detailed Description

Driver for reading and writing data from/to the FRAM.

Note
Please don't call any of those functions from an interrupt context

Typedef Documentation

typedef union _FRAMblockProtect FRAMblockProtect

Used by the FRAM_protectBlocks function to protect certain blocks of the FRAM from write operations. blockProtect Values: 0 = None, 1 = upper 1/4 protected, 2 = upper 1/2 protected, 3 = Entire FRAM protected
