Detailed Description

Exposes some boolean data types.

Values like 0xFFFFFFFF and 0xFF are used instead of 1 for reliability as this increases the number of bit-flips that turn a TRUE into FALSE. Preference is placed within ISIS drivers to use comparisons with FALSE.

     // Instead of
     if(flag == TRUE) {
         takeAction();
     }
     // We use
     if(flag != FALSE) {
         takeAction();
     }
     // Or
    if(flag) {
         takeAction();
     }
